## **접근 방법**

---

- 입력받은 정수 N 이상의 소수가 아닌 정수를 아무거나 출력하는 문제

- 조건은 N <= 1,000,000,000, 정답도 1,000,000,000 이하

- 1,000,000,000이 소수가 아니므로 **<U>입력에 상관없이 1,000,000,000 출력</U>**

<br/>

## **코드**

---

```java
public class Main {
	public static void main(String[] args) {
		System.out.println(1_000_000_000);
	}
}
```
